About the Weather Station Monitor

The Weather Station project was initiated to monitor the weather patterns for the local area. The monitor is located in rural NSW, Australia. Near the Murray River.

It is an example of using internet connectivity along with hardware and software design to accomplish an overall system designed, to suit an application.

The Weather Station Project is implemented using the Raspberry Pi, which is a credit card sized computer. RP Home Page The RP has aquired many fans, who are interested in experimenting with software and hardware.

The hardware is based on the very successful ARM chipset and, typically, the software is a standard Linux operating system, supported nearly all common programming and scripting languages.
